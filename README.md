# ClickIt - Repo Zip (Starter)

This archive contains a starter scaffold for ClickIt:
- backend (Node.js + Express + Prisma + Socket.IO)
- mobile (Expo React Native)
- admin (React + Vite + Tailwind)
- docker-compose for local development with Postgres and MinIO

Quickstart:
1. Install Docker & Docker Compose.
2. Copy `backend/.env.example` to `backend/.env` and edit values.
3. Run `docker-compose up --build`.
4. For mobile: `cd mobile && npm install && expo start`
5. For admin: `cd admin && npm install && npm run dev`
